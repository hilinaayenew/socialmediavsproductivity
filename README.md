# Student Productivity vs. Social Media Platforms usage

## Project Overview

This project investigates the relationship between daily social media consumption and actual productivity among students. Using a dataset of 30,000 behavioral records (filtered to 5,012 students), we explore whether higher usage time correlates with a decrease in productivity scores.

## Key Research Question

**"If social media usage is higher, does student productivity decrease?"**

## Findings

- **Correlation:** A strong negative correlation (approx. -0.62) between usage time and productivity.
- **Regression:** Every additional hour of social media is associated with a decrease of roughly 0.45 points on a 10-point productivity scale.
- **Nuance:** The study acknowledges the "Stress-Relief Hypothesis," suggesting that usage might also be a coping mechanism for existing academic stress.

## Files in this Repository

- `Hilina_Tiruneh_Ethiopia_FinalProject.ipynb`: The main Jupyter notebook containing the analysis, math, and visualizations.
- `social_media_vs_productivity.csv`: The dataset used for the analysis.
